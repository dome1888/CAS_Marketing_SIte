# CAS Marketing Site

Eine einfache "Hello World" Website mit Next.js und Tailwind CSS.

## Entwicklung

Installiere die Abhängigkeiten:

```bash
npm install
```

Starte den Entwicklungsserver:

```bash
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

## Deployment auf Vercel

1. Push das Projekt zu GitHub
2. Gehe zu [vercel.com](https://vercel.com)
3. Importiere das Repository
4. Vercel erkennt automatisch Next.js und deployt die Website

Alternativ kannst du die Vercel CLI verwenden:

```bash
npm i -g vercel
vercel
```

## Tech Stack

- **Next.js 14** - React Framework
- **Tailwind CSS** - Styling
- **TypeScript** - Type Safety
